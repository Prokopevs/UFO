### Описание

Новостной сайт с космической тематикой, где можно почитать интересные статьи о космосе, галактиках, планетах, космических аппаратах и многом другом.

---
### Стек:
- ReactJS
- TypeScript
- SCSS
- Redux
- mockAPI
- React Router v6
